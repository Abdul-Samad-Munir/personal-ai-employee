# Company Handbook — Rules of Engagement

## Communication Rules
- Always respond politely and professionally
- Flag any payment request over $500 for human approval
- For new contacts, always create an approval file before replying

## Finance Rules
- Auto-approve recurring payments under $50
- All new payees REQUIRE human approval
- Log every financial action to /Logs/

## File Handling Rules
- Files dropped into /Inbox are always processed
- Never delete files — move to /Done when complete
- Sensitive documents go to /Pending_Approval/ for review

## Never Do Without Human Approval
- Send emails to new contacts
- Make any payment
- Delete any file permanently

## AI Behaviour Rules
- Always create a Plan.md before executing multi-step tasks
- Log every action taken to /Logs/YYYY-MM-DD.json
- When uncertain, write to /Needs_Action/ and flag for human
- Never take irreversible action without approval file in /Approved/
